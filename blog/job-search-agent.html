Deep Dive: Job Search Agent AI Project

Job Search Agent is an open-source AI-powered tool that automates the process of searching and applying for jobs. Let's break down its architecture, core logic, and see how you can use or extend it for your own job-hunting automation.

Project Overview

  • Automates job search and application using AI agents
  • Uses LangChain, OpenAI, and Streamlit for the interface
  • Supports resume parsing, job scraping, and personalized cover letter generation

1. Installation & Setup

Clone the repo and install dependencies:

git clone https://github.com/surapuramakhil-org/Job_search_agent.git
cd Job_search_agent
pip install -r requirements.txt

Set your OpenAI API key in your environment:

export OPENAI_API_KEY=your-key-here

The agent uses LangChain and OpenAI's GPT models to generate tailored cover letters:

    prompt = f"Write a cover letter for {job['title']} using my resume: {resume_info}"
